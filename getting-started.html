Getting Started



Welcome to Subutai! You can start using Subutai either by:

  • Registering at the Subutai Bazaar, our social marketplace for user-administered P2P cloud services, and start creating environments on other users’ peers;
  • Installing the PeerOS in one or more computers and start offering your own computational resources for hire, barter or share.

In either case, you will also need to install the following companion software to ease your life:

  • The E2E (end-to-end encryption) plugin for your browser helps manage your SSH keys needed to ensure secure communications with Subutai components;
  • The P2P (peer-to-peer) daemon will allow your own PC to swarm up with the Subutai P2P cloud;
  • The Subutai Control Center in your desktop computer will give you quick access to your Peers, Environments and GoodWill balance.

Companion software


Browser plugin

The E2E browser plugin is available directly from the official stores of the main browsers - versions for Chrome, Firefox and Safari are currently offered. It will remove the hassle of managing SSH keys that are required to ensure secure communications within the Subutai platform.


P2P daemon

The P2P daemon allows your system to join the swarm to access private cloud environments. It is required if you want to access your Peers, containers and Environments via SSH or Remote Desktop from the Bazaar, Subutai Console or Control Center.

  • In order to download and install the P2P Daemon, please hop to the Downloads section (versions for Windows, Mac and Linux available) and follow the instructions.

Control Center

The Subutai Control Center allows you to access your Peers, containers and Environments via SSH or Remote Desktop, as well as checking and updating relevant local software and showing your GoodWill balance. In addition to that we have a file transfer system - users are able to easily upload and download their files to the remote cloud storage in a secure way.

  • In order to download and install the Subutai Control Center, please hop to the Downloads section (versions for Windows, Mac and Linux available).
  • Learn how to use the Control Center.

Install the Subutai P2P Daemon

The P2P daemon allows systems to join the swarm to access private cloud environments. We only support Debian based systems right now. So use apt install like so:

                                            sudo apt install /path/to/subutai-p2p.deb

Install the Subutai Control Center Application

Install the Control Center application to conveniently access environments from your desktop, manage your peers while showing you your Goodwill balance. Packages for Windows, Linux, and Mac OS are available. For Linux:

                                            sudo apt install /path/to/subutai-control-center.deb
