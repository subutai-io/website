Peer registration

Overview

Peer Registration module manages all local container hosts (create, destroy) and all peer relations (register, unregister and reject).


Peer registration steps:
  1. Go to Peer Registration module
  2. Press
  3. Write peer's IP address and Secret key-phrase then press Create button. A request will be sent to indicated peer.
  4. List of available Peers can be seen on Peers page of Subutai Hub

If peer registration is not approved, you need to wait until it gets a feedback: reject or register. If registered, the next step Environment creation using containers which will be used as a platform to execute all operations. If not, still an Environment can be created on a local machine using the same Environment creation chapter.
